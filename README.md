# Cypress, from Zero to the Cloud ☁️

This project was created during the **Cypress, from Zero to the Cloud** course to practice end-to-end (E2E) test automation with Cypress.

## Pre-requirements

It is required to have Node.js and npm installed to run this project.

> I used versions `v24.12.0` and `11.6.2` of Node.js and npm, respectively. I suggest you use the same or later versions.

## Installation

Run `npm install` (or `npm i` for the short version) to install the project dependencies.
 cy:open
## Tests

In this project, you can run the tests on a desktop or mobile viewport

### Desktop

Run `npm test` (or `npm t` for the short version) to execute the tests in headless mode a desktop viewpot.

Or, run `npm run cy:open` to open Cypress App on a desktop viewport.

### Mobile

Run `npm test:mobile` (or `npm t` for the short version) to execute the tests in headless mode a mobile viewpot.

Or, run `npm run cy:open:mobile` to open Cypress App on a mobile viewport.

## About

This repository contains the exercises and examples developed throughout the course, covering topics such as:

- Cypress setup and configuration
- End-to-end testing
- Assertions and validations
- Test organization
- Best practices for test automation
- Running tests locally and in CI environments
